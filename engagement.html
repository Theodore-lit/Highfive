Votre futur métier

Nos formations conduisent aux fontions de Développeur codeur qui recouvre des réalités très différentes en fonction:

  • de la finalité du développeur front end (développe les éléments que les internautes voient sur leur écran) : création de sites, d'application, de programmation / back end : (développe ce que les internautes ne voient pas) : un serveur, une application, une base de données, le mode de paiement sur un site e-commerce etc / full-stack (front et back end)
  • des languages utilisés et des outils à maitrîser : HTML, CSS, JavaScript, Wordpress, PHP, MySQL, SQL, C#, C++, Python, Linux, Dupral, Magento... Framework: React, Angular, Vue.js, Symfony, Laravel;
  • du type de client pour lequel le « Développeur Codeur » intervient : client interne à une organisation, prestataire informatique en sous-traitance (cabinet conseil, agence…) et très rarement de clients utilisateurs finaux).

Avec le soutien d'HIGHFIVE University, vous accédez à un niveau de « maitrise du code » qui vous permet d'exercer la plupart des fonctions d’un développeur codeur

  • Dévéloppeur/ Intégrateur Web
  • Webmaster
  • Développeur front end
  • Développeur back end
  • Concepteur Dévéloppeur Web
  • Chef Projet Web
  • Développeur Full Stack
  • Data Analyst
  • Dévéloppeur mobile
  • Architecte d'application Web
  • Consulteur IT
  • Ingénieur d'étude et Développement
  • Data Scientist
